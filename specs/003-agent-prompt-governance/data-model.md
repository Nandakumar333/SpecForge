## Domain Entities

### PromptThreshold

```python
@dataclass(frozen=True)
class PromptThreshold:
    """A single numeric or string threshold within a rule."""
    key: str    # e.g., "max_class_lines", "min_coverage_pct"
    value: str  # raw string value (e.g., "200", "80")
```

### PromptRule

```python
@dataclass(frozen=True)
class PromptRule:
    """An individual enforceable constraint within a governance file."""
    rule_id: str                          # e.g., "SOLID-001", "SEC-003"
    title: str                            # e.g., "Single Responsibility"
    severity: str                         # "ERROR" | "WARNING"
    scope: str                            # e.g., "classes, methods"
    description: str                      # full rule text (MUST / MUST NOT)
    thresholds: tuple[PromptThreshold, ...]
    example_correct: str                  # code snippet (✅)
    example_incorrect: str                # code snippet (❌)
```

**Validation rules**:
- `rule_id` MUST match pattern `^[A-Z]+-\d{3}$`
- `severity` MUST be one of `{"ERROR", "WARNING"}`
- `description` MUST contain at least one of: `MUST`, `MUST NOT`, `is prohibited`
- `thresholds` MAY be empty (non-numeric rules)

### PromptFileMeta

```python
@dataclass(frozen=True)
class PromptFileMeta:
    """Parsed metadata from a governance file's ## Meta section."""
    domain: str      # e.g., "backend"
    stack: str       # e.g., "dotnet" | "agnostic"
    version: str     # e.g., "1.0"
    precedence: int  # 1 (highest) to 7 (lowest); maps to PRECEDENCE_ORDER index
    checksum: str    # SHA-256 hex of original rendered content
```

### PromptFile

```python
@dataclass(frozen=True)
class PromptFile:
    """A single parsed governance file for one domain."""
    path: Path
    meta: PromptFileMeta
    rules: tuple[PromptRule, ...]
    raw_content: str  # original Markdown text (for PromptContextBuilder injection)
```

### PromptSet

```python
@dataclass(frozen=True)
class PromptSet:
    """All 7 governance files for a project. Return type of PromptLoader."""
    files: dict[str, PromptFile]  # domain name → PromptFile
    precedence: list[str]         # PRECEDENCE_ORDER constant value
    feature_id: str               # passthrough from load_for_feature()
```

### ConflictEntry

```python
@dataclass(frozen=True)
class ConflictEntry:
    """A single detected threshold conflict between two governance files."""
    threshold_key: str      # e.g., "max_class_lines"
    rule_id_a: str          # rule ID in domain_a
    domain_a: str           # e.g., "architecture"
    value_a: str            # e.g., "50"
    rule_id_b: str          # rule ID in domain_b
    domain_b: str           # e.g., "backend"
    value_b: str            # e.g., "30"
    winning_domain: str     # domain_a | domain_b | "AMBIGUOUS"
    winning_value: str      # value of winning domain, or "" if ambiguous
    is_ambiguous: bool      # True when domain_a and domain_b have equal precedence
    suggested_resolution: str  # one-line human-readable fix instruction
```

### ConflictReport

```python
@dataclass(frozen=True)
class ConflictReport:
    """Output of PromptValidator.detect_conflicts()."""
    conflicts: tuple[ConflictEntry, ...]
    has_conflicts: bool  # True iff conflicts is non-empty
```

## Governance File Format (canonical structure)

Every governance file generated by SpecForge MUST follow this structure exactly.
The parser depends on heading names and indentation conventions.

```markdown
<!-- Generated by SpecForge — do not edit manually above the Rules section -->

# {domain} Governance Prompt

## Meta
domain: {domain}
stack: {stack}
version: 1.0
precedence: {1-7}
checksum: {sha256hex}

## Precedence
This file occupies position {N} in the conflict-resolution hierarchy:
`security (1) > architecture (2) > backend/frontend/database (3) > testing (4) > cicd (5)`

When this file conflicts with a higher-precedence file, the higher-precedence
value applies. When this file conflicts with an equal-priority file, the conflict
is flagged as AMBIGUOUS and requires manual resolution.

## Rules

### {CATEGORY}-{NNN}: {Title}
severity: ERROR
scope: {scope description}
rule: {Full MUST/MUST NOT statement.}
threshold: {key1}={value1}, {key2}={value2}
example_correct: |
  {code snippet}
example_incorrect: |
  {code snippet}
```

**Parser invariants**:
- `## Meta` MUST appear before `## Rules`
- Each rule block begins with `### {RULE-ID}: {Title}` (exactly 3 `#` symbols)
- All key-value fields within a rule are single-line `key: value` pairs
- Multi-line values use `|` continuation with 2-space indent on subsequent lines
- `threshold:` line contains comma-separated `key=value` pairs

## Config Constants (additions to `config.py`)

```python
# Governance domain list (insertion order = file generation order)
GOVERNANCE_DOMAINS: list[str] = [
    "architecture",
    "backend",
    "frontend",
    "database",
    "security",
    "testing",
    "cicd",
]

# Conflict resolution order (index 0 = highest priority)
PRECEDENCE_ORDER: list[str] = [
    "security",       # 1 — always wins
    "architecture",   # 2 — cross-cutting rules
    "backend",        # 3 — equal priority group
    "frontend",       # 3 — equal priority group
    "database",       # 3 — equal priority group
    "testing",        # 4
    "cicd",           # 5
]

# Domains with equal precedence — intra-group conflicts are AMBIGUOUS
EQUAL_PRIORITY_DOMAINS: frozenset[str] = frozenset({"backend", "frontend", "database"})

# Domains that are always generated as agnostic (no stack suffix in filename)
AGNOSTIC_GOVERNANCE_DOMAINS: frozenset[str] = frozenset({"architecture", "security"})

# Governance file path pattern
GOVERNANCE_FILE_PATTERN: str = "{domain}.{stack}.prompts.md"
GOVERNANCE_AGNOSTIC_FILE_PATTERN: str = "{domain}.prompts.md"

# Config file path (relative to project root)
SPECFORGE_CONFIG_FILE: str = ".specforge/config.json"
```

### `src/specforge/core/prompt_validator.py`

```python
class PromptValidator:
    def detect_conflicts(
        self, prompt_set: PromptSet
    ) -> ConflictReport: ...
    # Scans all rules+thresholds, returns ConflictReport.
    # Never raises — always returns a ConflictReport (possibly empty).

    def _build_threshold_index(
        self, prompt_set: PromptSet
    ) -> dict[str, list[tuple[str, str, str]]]: ...
    # threshold_key → [(domain, rule_id, value), ...]

    def _find_conflicts(
        self,
        index: dict[str, list[tuple[str, str, str]]]
    ) -> list[ConflictEntry]: ...

    def _build_suggestion(
        self, winner: tuple, loser: tuple, is_ambiguous: bool
    ) -> str: ...
```

### `src/specforge/core/prompt_context.py`

class PromptContextBuilder:
    def build(

    # Returns all 7 files' raw_content joined in PRECEDENCE_ORDER order.
    # task_domain is used only to put the most relevant domain FIRST
    # in the output string — all 7 files are always included.

## State Machine: `specforge init` with governance file generation

```text
┌─────────────────────────────────────────────────────────────────┐
│                     specforge init myapp                        │
└────────────────────────────┬────────────────────────────────────┘
                             │
                    ┌────────▼─────────┐
                    │  Detect stack    │ ← --stack flag OR StackDetector
                    │  (FR-016)        │
                    └────────┬─────────┘
                             │
                    ┌────────▼─────────┐
                    │  Build scaffold  │ ← existing ScaffoldPlan
                    │  plan            │
                    └────────┬─────────┘
                             │
                    ┌────────▼──────────────┐
                    │  PromptFileManager    │
                    │  .generate(config)    │
                    │  ─ resolve templates  │
                    │  ─ render Jinja2      │
                    │  ─ compute checksums  │
                    │  ─ write 7 files      │
                    └────────┬──────────────┘
                             │
                    ┌────────▼─────────┐
                    │  Write           │
                    │  config.json     │ ← {stack, project_name, version}
                    └────────┬─────────┘
                             │
                    ┌────────▼─────────┐
                    │  git commit      │ ← "chore: init specforge scaffold"
                    └──────────────────┘
```

## State Machine: `specforge init --force` (customization preservation)

```text
For each governance file in GOVERNANCE_DOMAINS:
    │
    ├── file exists?
    │       NO  → generate fresh (always)
    │       YES → is_customized(file)?
    │                   YES → SKIP (preserve user edits) [FR-014]
    │                   NO  → regenerate (overwrite default content)
    │
    └── continue to next domain
```
