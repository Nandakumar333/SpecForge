## ARCHITECTURE NOTES

The codebase follows:
1. **Frozen dataclasses** for immutability
2. **Pure functions** for state transitions
3. **Result[T, E] monad** for error handling (not exceptions)
4. **Atomic writes** with temp files and os.rename()
5. **ISO 8601 timestamps** for all temporal data
6. **Schema versions** for forward compatibility
7. **Tuples for immutable collections** (not lists)
8. **Type hints throughout** (TYPE_CHECKING for conditional imports)
9. **pytest + syrupy** for testing with snapshot validation
